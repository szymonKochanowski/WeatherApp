Project name: WeatherApp

Purpose of project: Learn how to connect with external API.

Description: Project allows user to get current weather based on provided cityName.
App based on city name is taking geolocation based on city name based on external API and them show current weather also based on external API,
like as current, max and min temperature, wind speed, pressure, humidity, clouds and date time.
App will save geolocation for specified city in database in table geolocation if not exist in db and save all weathers in db
in table weather for all calls.

Used:
- Java version 19,
- Spring Boot 3,
- Liquibase,
- Mapstruck,
- MySQL DB,
- Lombok,
- JSON,
- Mockito 5 (JUnit Jupiter).
